Über diese Seite:

Der DHBW Web Timer trackt wie viel Zeit du im Internet verbringst und präsentiert dir informative und intuitive Statistiken über dein Browserverhalten.

Home Page

Auf der Homepage siehst du dein Verhalten am aktuellen Tag

Auf der Homepage siehst du dein Verhalten am aktuellen Tag


  • Hover über den Doughnut um detaillierte Infos über dein Verhalten auf der aktuellen Seite zu erfahren
  • Die Tabelle zeigt alle Infos auf einen Blick
 Statistik: Tag

Browsingverhalten an einem wählbaren Tag

Auf der Tag Seite siehst du dein Verhalten an einem bestimmten Tag


  • Nutze den Datepicker oben, um deinen Tag auszuwählen
  • Restliche Funktionen
 Statistik: Woche Browsingverhalten an einer Woche oder vielen Wochen

Auf der Wochen Seite siehst du dein Verhalten an einer bestimmten Woche


  • Klicke auf "Woche hinzufügen" um mehrere Wochen miteinander zu vergleichen. Nutze den date-picker um deine gewünschte Woche auszuwählen
  • Entferne Wochen aus der Statistik mit dem Mülleimer icon
  • Die Balken zeigen dein Verhalten an den einzelnen Tagen der Wochen und sind nach Seiten aufgeteilt
 Statistik: Monat Browsingverhalten an einem Monat oder vielen Monaten

Auf der Monatsseite siehst du dein Verhalten an einer oder vielen bestimmten Monaten


  • Klicke auf "Monat hinzufügen" um mehrere Monate miteinander zu vergleichen. Nutze den date-picker um deinen gewünschten Monat auszuwählen
  • Entferne Monate aus der Statistik mit dem Mülleimer icon
  • Die Line-Chart zeigen dein Verhalten an den einzelnen Tagen der Monate an
 Statistik: Gesamt Browsingverhalten über den gesamten erfassten Zeitraum

Auf der Gesamtseite siehst du dein Verhalten über den kompletten Zeitraum


  • Die Funfacts oben zeihen *witzige* Informationen über dein Browserverhalten
  • Hover über den Doughnut um detaillierte Infos über dein Verhalten auf der aktuellen Seite zu erfahren
  • Die Tabelle zeigt alle Infos auf einen Blick


Installation:

  1. Downloade die aktuelle Version der Chrome Extension über den Download Button unten
  2. Öffne den Chrome Browser und navigiere nach chrome://extensions/ und schalte den Entwicklermodus rechts oben ein
  3. Klicke links oben auf "Entpacke Erweiterung laden"
  4. Browser zum src Folder des entpackten Chrome Extension die du gerade heruntergeladen hast und bestätige

Die Extension sollte jetzt rechts oben in der Extensionbar des Chome browsers verfügbar sein
